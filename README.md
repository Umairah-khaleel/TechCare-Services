# Tech Care Service Mobile Application

A comprehensive Android mobile application built with Java, designed to bridge the gap between technical service providers (admins) and clients (customers). The project follows a structured data-driven approach to streamline booking operations, price visibility, and client support workflows.

---

## 🚀 Core Features

### 👥 Dual-Role User Authentication
* **Customer Dashboard:** A clean, intuitive user interface for navigating maintenance resources, checking transparent pricing lists, and booking immediate repairs.
* **Admin Dashboard:** A robust control center allowing administrators to manage Incoming service tickets, handle active bookings, and dynamically oversee operational logs.
* **Secure Registration:** Secure signup flows to safely register new clients into the platform database.

### 🛠️ Client Support & Engagement
* **Dynamic Price List:** An up-to-date catalog of technical care services and upfront estimated pricing packages.
* **Maintenance Tips & FAQs:** Proactive technical advice, device optimization tips, and an automated FAQ module to decrease service ticket repetition.
* **Booking Management System:** A seamless workflow for clients to request hardware or software care and track current repair status.

---

## 📊 Development Methodology
The design and deployment of this application were strictly guided by the **CRISP-DM Framework** (CRoss-Industry Standard Process for Data Mining). This structured approach ensured that features like the booking patterns and the dynamic pricing modules were directly optimized to resolve core business pain points and customer needs.
